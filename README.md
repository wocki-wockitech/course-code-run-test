# Playground & Code Challenges — Demo Course

Демо-курс для тестирования и демонстрации всех типов code-блоков платформы StudyMe.

## Содержание

### Block 1: Playground Basics
- Hello Playground — запуск кода в браузере
- Multi-file — пакетная структура
- Languages & Versions — выбор языка и версии

### Block 2: Coding Challenges
- Basic Challenge — задача с автотестами (go test)
- Challenge with Hints — подсказки
- SQL Challenge — проверка SQL-запросов

### Block 3: Advanced Modes
- Vet & Lint — статический анализ
- Race Detector — проверка конкурентности (go test -race)
- Benchmarks — проверка производительности

## Для авторов

Этот курс служит примером использования:
- `> [!sandbox] lang` — встроенная песочница
- `> [!challenge] slug` — coding challenge с автотестами
- `test_mode: vet | test_race | bench` — расширенные режимы проверки
- `comparison.mode` — SQL grading modes
