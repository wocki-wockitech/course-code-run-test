# Привет, Playground

Playground — встроенная песочница для запуска кода прямо в браузере. Код исполняется в изолированном контейнере (gVisor) — безопасно и без установки чего-либо на компьютер.

## Как это работает

1. Пишете код в редакторе
2. Нажимаете **Run**
3. Видите вывод в реальном времени (SSE-стрим)

Под капотом: ваш код отправляется на сервер → создаётся эфемерный под в Kubernetes с RuntimeClass gVisor → код компилируется и запускается → stdout стримится обратно.

## Попробуйте

> [!sandbox] go

Измените программу — выведите своё имя. Нажмите Run.

## Ограничения sandbox

- **Нет сети** — нельзя скачать пакеты или обратиться к внешним API
- **Таймаут** — программа убивается через 10 секунд
- **Память** — ограничена (1Gi в dev, меньше в проде)
- **Файлы** — до 20 файлов, до 64KB каждый

Эти ограничения — by design. Sandbox для обучения, не для production-разработки.

## Что видно в выводе

Playground показывает стадии выполнения:
- **compiling** — Go компилирует ваш код
- **running** — программа запущена
- Финальный статус: `done` (exit 0), `timeout` (превышен лимит), или ошибка компиляции
