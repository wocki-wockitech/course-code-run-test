# Race Detector

Go имеет встроенный детектор гонок данных (data race detector). Это уникальная фича — ни один другой мейнстрим-язык не имеет аналога из коробки.

## Что такое data race

Data race — когда две горутины одновременно обращаются к одной переменной и хотя бы одна из них пишет. Результат непредсказуем.

```go
// DATA RACE:
var counter int

go func() { counter++ }()
go func() { counter++ }()
// counter может быть 1 или 2 — неопределённое поведение
```

## Как работает -race

`go test -race` инструментирует код — добавляет проверки на каждый доступ к памяти. Если обнаружена гонка — тест падает с подробным стек-трейсом обоих обращений.

**Цена:** потребление памяти ×5-10, скорость ×2-10. Поэтому race detector используется при тестировании, не в проде.

## Как используется в challenge

```yaml
# challenge.yaml
test_mode: test_race    # go test -race
limits:
  memory_mb: 2048       # race detector жрёт память
```

Вердикт: если race detector нашёл гонку — fail (даже если тесты "проходят").
