# Challenge с подсказками

Автор может добавить подсказки — студент открывает их по одной, если застрял.

## Подсказки (hints)

В `challenge.yaml`:
```yaml
hints:
  - "Используй sync.Mutex для защиты shared state"
  - "Lock() перед записью, Unlock() после"
  - "defer mu.Unlock() — идиоматичный паттерн"
```

Студент видит кнопку "Подсказка" → раскрывает по одной. Это НЕ штрафует (в отличие от code review), но можно учитывать в SR-повторении (качество recall ниже если использовал подсказки).
